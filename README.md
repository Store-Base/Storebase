🛒 Sistema de Gerenciamento de Loja
📌 Sobre o Projeto

O Sistema de Gerenciamento de Loja é uma aplicação desenvolvida com o objetivo de automatizar e organizar as operações comerciais de uma loja.

O sistema permite o controle completo de:

Produtos
Clientes
Orçamentos
Vendas
Estoque
Funcionários
Relatórios

A proposta é substituir processos manuais por uma solução digital eficiente, reduzindo erros e aumentando a produtividade.

🎯 Objetivo

Desenvolver um sistema capaz de gerenciar todas as atividades comerciais de uma loja, proporcionando:

Organização dos dados
Agilidade no atendimento
Controle de estoque em tempo real
Suporte à tomada de decisões
